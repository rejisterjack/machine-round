# Demo video script (~3 minutes)

Record in 1080p, incognito browser, with mic audio. Edit for zero dead air.

| Time | Section | On screen | Narration |
|---|---|---|---|
| 0:00–0:20 | Hook | Your face or title card | "Three days before this hackathon I sat in a real AI hiring screen. No human in the room — just a model scoring clarity, structure, and depth." |
| 0:20–0:50 | Problem | Split: human prep sites vs AI screen | "Every prep tool trains you for a human interviewer. AI evaluators listen for different signals — specifics, structure, no rambling." |
| 0:50–1:10 | Landing | namaste-machine-round URL | "Namaste Machine Round is practice for the machine. Judges can preview a sample report without signing in." |
| 1:10–2:20 | Live product | Role select → voice session | "I pick Full-Stack Engineer, join the voice room, and answer live. Watch this follow-up — it references what I just said about caching." **(Pause on adaptive follow-up.)** |
| 2:20–2:50 | Report | Readiness report + share | "When the round ends, a separate evaluator agent scores the transcript. Per-answer breakdown, red flags, improvement actions." |
| 2:50–3:00 | Payoff | Share link + PDF + replay | "Share the report, download PDF, or send judges a replay link — all from one session." |

## Recording checklist

- [ ] Production URL loads cold in incognito
- [ ] Google sign-in works
- [ ] One clearly adaptive follow-up captured on mic
- [ ] Report generates in under 10 seconds
- [ ] Share link opens without auth
- [ ] Export PDF from shared report page

## B-roll shots (optional)

- History page with filters
- Screen snapshot timeline on replay
- CI badge / README architecture diagram
